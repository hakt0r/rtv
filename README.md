## RTV - The pocket Radio and TV station

Based upon:

  * [rex](https://github.com/hakt0r/rex)
  * [liquidsoap](http://savonet.sf.net)

### Installation

    $ sudo npm install -g git://github.com/hakt0r/rtv.git

### CLI usage

    $ rtv -i

  * -i : start in interactive ncurses mode
